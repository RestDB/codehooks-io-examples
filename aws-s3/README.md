# AWS S3 v3 example

Example app to create an API for upload and download of binary files to AWS S3

## Install
```bash
npm install codehooks -g
```

Login to account

```bash
coho create mys3project
cd mys3project

npm init es6 -y
npm install codehooks-js
npm install @aws-sdk/client-s3
```

Edit codefiles.

```bash
coho deploy
```
